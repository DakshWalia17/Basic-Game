# Mini Python Games

A simple command-line application that allows you to play two classic games: **Stone-Paper-Scissors** and a **Dice Roll Game**. Built with foundational Python concepts, this project is easy to read, understand, and run.

## Features

- **Stone-Paper-Scissors**: Play the classic game against the computer.
- **Dice Roll Game**: Roll a virtual 6-sided die against the computer to see who gets the higher number.
- **Interactive Menu**: A never-ending menu loop that lets you play multiple rounds or switch games until you choose to exit.

## File Structure

- `main.py`: The main entry point of the program. It handles the interactive menu and imports the game logic.
- `rps.py`: Contains the logic for the Stone-Paper-Scissors game.
- `dice.py`: Contains the logic for the Dice Roll Game.

## Requirements

- Python 3.x

No external libraries are required as the project only uses the built-in `random` module.

## How to Run

1. Open your terminal or command prompt.
2. Navigate to the folder containing the project files.
3. Run the following command:

```bash
python main.py
```

4. Follow the on-screen prompts to choose a game and start playing!
